SAMIHA ZEINEDDINE LIFE COACH PLATFORM - INSTALLATION GUIDE
================================================================================

PROJECT OVERVIEW:
This is a complete, fully functional static website for Samiha Zeineddine's 
life coaching services. Built with pure HTML, CSS, and JavaScript - no backend 
required for basic functionality.

================================================================================
QUICK START (3 STEPS)
================================================================================

1. EXTRACT THE ZIP FILE
   - Unzip "newsamiha-complete.zip" to your desired location
   - You'll see a folder called "newsamiha-static"

2. START A LOCAL SERVER
   Option A - Python 3:
   - Open terminal/command prompt in the newsamiha-static folder
   - Run: python3 -m http.server 8000
   
   Option B - Python 2:
   - Run: python -m SimpleHTTPServer 8000
   
   Option C - Node.js (if installed):
   - Run: npx http-server

3. OPEN IN BROWSER
   - Go to: http://localhost:8000
   - You'll see the landing page
   - Click "Login" to access the user dashboard

================================================================================
CUSTOMIZATION
================================================================================

CHANGE COLORS:
1. Open: css/style.css
2. Find the color definitions (look for hex colors like #8B7355)
3. Replace with your preferred colors

CHANGE TEXT/CONTENT:
1. Open any .html file in a text editor
2. Find the text you want to change
3. Edit and save
4. Refresh browser to see changes

ADD TRANSLATIONS:
1. Open: js/i18n/en.json (or ar.json, fr.json)
2. Add new key-value pairs
3. Update HTML to use the translation key

CHANGE IMAGES:
1. Replace image files in: assets/images/
2. Keep the same filename or update HTML references

DEPLOYMENT OPTIONS
================================================================================

OPTION 1 - GITHUB PAGES (FREE):
1. Create a GitHub account (if you don't have one)
2. Create a new repository named "newsamiha"
3. Upload all files to the repository
4. Go to Settings > Pages > Select "main" branch
5. Your site will be live at: https://yourusername.github.io/newsamiha

OPTION 2 - NETLIFY (FREE):
1. Go to netlify.com
2. Click "New site from Git"
3. Connect your GitHub repository
4. Netlify will automatically deploy
5. Your site will be live at a netlify.app URL

OPTION 3 - TRADITIONAL WEB HOSTING:
1. Get a web hosting account (GoDaddy, Bluehost, etc.)
2. Use FTP/SFTP to upload all files
3. Set index.html as the default document
4. Your site will be live at your domain

OPTION 4 - LOCAL NETWORK:
1. Run the HTTP server (see Quick Start)
2. Find your computer's IP address
3. Share the IP address with others on your network
4. They can access at: http://YOUR_IP:8000

================================================================================
TROUBLESHOOTING
================================================================================

PROBLEM: "Port 8000 already in use"
SOLUTION: Use a different port: python3 -m http.server 9000
          Then visit: http://localhost:9000

PROBLEM: Language selector not working
SOLUTION: Make sure you're running a server (not opening file directly)
          Direct file access has CORS restrictions

PROBLEM: Images not loading
SOLUTION: Check that assets/images/ folder exists with all images
          Verify image filenames in HTML match actual files

PROBLEM: Styles look broken
SOLUTION: Clear browser cache (Ctrl+Shift+Delete or Cmd+Shift+Delete)
          Make sure all CSS files are in css/ folder

PROBLEM: Links not working
SOLUTION: Verify all .html files are in the same directory
          Check that file names match exactly (case-sensitive)
